== Installation ==
- Install & Activate Bitdroplet SIP Plugin
- Go to Widgets section under Appearance Tab, then drag & drop Bitdroplet SIP widget to either Sidebar section, footer section or wherever you want to show this widget.
- The Widget allows you to configure 2 things.
   - A unique tag for tracking referral. Any traffic that bitdroplet.com receives via this widget will be tracked based on this Tracking Code. __Get your Unique Referral code from [here](https://bitdroplet.com/app/#/profile/widget)__.
		e.g __*BITDROP18333_WIDGET*__
   - You could also customise the Primary color of the widget according to your website. Please provide hex color code only __WITHOUT *Hash (#)*__ eg: 574B90 
- Save the configs and That's it. The Bitdroplet Widget should appear on your wordpress website.
